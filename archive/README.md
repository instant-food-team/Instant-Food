# ARCHIVE

本目录用于归档当前项目中的临时调试产物，不存放正式 UI 页面、正式文档或正式脚本。

当前归档规则：
- `browser-profiles/`
  - 存放浏览器自动化、联调、自检过程中生成的临时 Edge profile。
- `misc/`
  - 存放不再作为正式开发文档使用、但暂时不删除的历史杂项文件。

明确不归档到这里的内容：
- `frontend/`
  - 正式页面原型与小程序前端目录。
- `docs/`
  - 项目治理文档、技术架构文档、交接文档。
- `scripts/`
  - 项目辅助脚本。
- `frontend/assets/`
  - 背景素材池或设计素材。
- `tests/`
  - 页面自检截图、状态截图、人工验收图。
- `output/`
  - 工具或流程生成的输出目录，后续单独判断是否需要继续拆分。

后续整理原则：
- 根目录只保留正式项目目录和少量必要入口文件。
- 所有当前活动中的 `tmp-*` 优先归到 `tmp/`。
- 所有截图、自检图、人工验收图优先归到 `tests/`。
- 如果某个文件未来会被页面直接引用，就不要放进 `archive/`。
